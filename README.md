 # BTC Oracle Market

An always-open Bitcoin prediction market built on OPNet.

## What it does
Users can predict whether the BTC price is above a fixed target using BTC.
The market never closes and uses oracle-based BTC/USD pricing.

## Features
- BTC / rBTC only
- Always-open prediction market
- Dynamic odds based on pool sizes
- Minimum $5 deposit and withdrawal (BTC value)
- Oracle-based price verification
- Withdrawals may take up to 24 hours (notice only)

## Tech
- OPNet smart contracts
- On-chain BTC price oracle
